NutriCare

AI-powered personalized nutrition guidance using Machine Learning prediction and LLM-based contextual advice generation.

Problem Statement

Most online nutrition platforms provide generic dietary advice that does not consider individual health conditions, goals, or activity levels. Chronic illness patients require structured, condition-aware guidance, but current solutions lack contextual personalization and AI-driven decision support.

Technical Methodology

User Profiling

Structured input collection: Age, Weight, Activity Level, Goal, Health Conditions.

ML Prediction

Multi-output regression model predicting Calories, Protein, Carbs, and Fat.

LLM Integration

OpenRouter Mistral model with structured prompt engineering and backend response parsing.

Results & Analysis

Average ML Prediction Accuracy ≈ 70% across macro targets. Model trained using ensemble techniques and serialized via joblib.

*AI-assisted system. Not a substitute for professional medical advice.*
